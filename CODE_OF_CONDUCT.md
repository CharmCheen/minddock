# Code of Conduct

## Our Standard

We expect contributors to communicate clearly, act respectfully, and keep technical discussion focused on the work.

## Expected Behavior

- Use welcoming and professional language.
- Accept technical feedback in good faith.
- Focus on evidence, reproducibility, and maintainability.
- Respect differing experience levels and viewpoints.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Publishing private information without permission.
- Disruptive behavior that blocks collaboration.

## Enforcement

Project maintainers may remove comments, commits, or contributions that violate this code of conduct.
