# Frontend

Vite + React + TypeScript frontend for MindDock.

## Quick Start

```bash
npm install
npm run dev
```

## Structure

```
src/
  core/       # Core utilities, API client, auth
  features/   # Feature modules (chat, workspace, etc.)
  lib/        # Shared UI components, hooks, utils
public/       # Static assets
```
